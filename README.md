# Data Jobs Market Analysis & Interactive Dashboards

This directory contains data analytics projects built in both **Excel** and **Power BI** using data job market datasets.

The projects explore questions such as:

- Which data jobs pay the most?
- What skills are most in demand?
- Do higher-paying jobs require more skills?
- Which skills are associated with specific data careers?
- How do salaries and job opportunities vary across roles and regions?

The goal is to help aspiring data professionals better understand the job market and make informed decisions about career paths and skill development.
